Introdução ao HTML

O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto).

Importante: HTML não é uma linguagem de programação, é uma linguagem de marcação usada para organizar e estruturar o conteúdo de uma página para que o navegador saiba como exibi-lo.

Origem do HTML

O HTML foi criado por Tim Berners-Lee em 1991. O objetivo era permitir a conexão entre documentos em diferentes instituições, formando a base da web como conhecemos hoje.

Nome do arquivo principal

Por convenção, o arquivo principal de um site costuma se chamar:

index.html

Como funcionam as tags?

O HTML é composto por tags (etiquetas). Exemplo básico: 

<p>Este é um parágrafo</p>
  • <p> → tag de abertura
  • </p> → tag de fechamento
  • O texto fica entre as duas tags

Dica: Foque em entender como as tags funcionam em vez de decorá-las.

Estrutura básica de uma página HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Título da página</title>
  </head>
  <body>
    <h1>Meu site</h1>
    <p>Olá mundo!</p>
  </body>
</html>

Explicando cada parte:

  • <!DOCTYPE html> → informa ao navegador que estamos usando HTML5
  • <html> → envolve todo o conteúdo da página
  • <head> → contém informações não visíveis ao usuário (título da aba, configurações, links para CSS)
  • <title> → define o nome que aparece na aba do navegador
  • <body> → contém tudo que aparece na tela (textos, imagens, títulos, links)

Elementos comuns

  • <h1>, <h2> → títulos
  • <p> → parágrafos
  • <img> → imagens
  • <a> → links
  • <footer> → rodapé da página

HTML5

O HTML5 é a versão moderna do HTML (em uso desde 2014). Ele trouxe melhorias como:

  • Melhor organização semântica
  • Novas tags, como <video>

Criando links

Para criar um link, usamos a tag <a>:

<a href="https://exemplo.com">Clique aqui</a>

Dica prática (Visual Studio Code)

No editor Visual Studio Code, digite:

html5

e pressione Enter para gerar automaticamente a estrutura básica da página.

Conclusão

  • HTML é a base de qualquer site
  • Ele organiza o conteúdo da página
  • Aprender HTML é o primeiro passo para desenvolvimento web
